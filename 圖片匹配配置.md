## 🔧 匹配演算法

### 主要匹配邏輯
```javascript
// 1. 直接名稱匹配
if (productImageMapping[product.name]) {
    return productImageMapping[product.name];
}

// 2. 模糊匹配演算法
function findBestMatch(productName, availableNames) {
    const productKeywords = productName.toLowerCase().split(/[\s\-_]+/);
    
    let bestMatch = null;
    let bestScore = 0;
    
    for (const availableName of availableNames) {
        const availableKeywords = availableName.toLowerCase().split(/[\s\-_]+/);
        
        let matchCount = 0;
        for (const keyword of productKeywords) {
            if (availableKeywords.some(ak => ak.includes(keyword) || keyword.includes(ak))) {
                matchCount++;
            }
        }
        
        const score = matchCount / Math.max(productKeywords.length, availableKeywords.length);
        
        if (score > bestScore && score > 0.3) { // 30%相似度門檻
            bestScore = score;
            bestMatch = availableName;
        }
    }
    
    return bestMatch;
}
```

## 📝 新增商品圖片步驟

### 1. 準備圖片檔案
- 檔名格式：`品牌_型號_規格_WATERMARKED.jpg`
- 多圖片編號：`-1`, `-2`, `-3`, `-4`
- 封面圖片：通常使用 `-1` 或沒有編號

### 2. 放置圖片檔案
```bash
# 放到對應類別資料夾
cp 新圖片.jpg images/相對應類別/
```

### 3. 更新匹配配置
編輯 `update-product-images.js` 中的 `productImageMapping`：
```javascript
"新商品名稱": {
    images: {
        cover: "./images/類別/新圖片-1.jpg",
        gallery: [
            "./images/類別/新圖片-2.jpg",
            "./images/類別/新圖片-3.jpg"
        ]
    }
}
```

### 4. 執行匹配
```javascript
// 在瀏覽器控制台執行
updateProductImages();
```

## 🔍 故障排除

### 圖片不顯示
1. 檢查檔案路徑是否正確
2. 確認圖片檔案存在
3. 檢查檔名大小寫
4. 驗證圖片格式支援

### 匹配失敗
1. 檢查商品名稱是否完全一致
2. 嘗試模糊匹配
3. 檢查品牌關鍵字
4. 手動添加匹配規則

### 效能問題
1. 壓縮大型圖片檔案
2. 使用WebP格式
3. 實施懶載入
4. 減少同時載入的圖片數量
